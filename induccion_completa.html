Inducción Completa

La inducción completa es un método de prueba utilizado en matemáticas para demostrar que una proposición es verdadera para todos los números naturales. Se basa en dos pasos fundamentales:

  1. **Base de Inducción**: Se verifica que la proposición sea verdadera para un valor inicial, generalmente n = 1.
  2. **Paso Inductivo**: Se supone que la proposición es verdadera para n = k, y se demuestra que también es verdadera para n = k + 1.

Si ambos pasos son válidos, se concluye que la proposición es verdadera para todos los números naturales.
